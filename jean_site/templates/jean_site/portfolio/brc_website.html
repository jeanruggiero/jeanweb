My latest project is a complete redesign of the website I made last year for the Boealps Basic Rock Class. In my second year as head instructor, I identified a need for a website to better organize and distribute course content, and for students to interact with each other during the class. I created a basic website using Django that did the job for a year, but it had a lot of usability issues and the visual design needed to be improved.

This year, I'm working on revamping the website. The backend is being transformed into a Django REST API that adheres to the OpenAPI specification. The API specification is out and can be viewed on Swagger Hub.

The front end will be a React app and will be built according to the design specification I created in Sketch. A clickable prototype of the design can be viewed using the button up top.
